Detailed Description

class to navigate in the raw data buffer parse the header and send the payload as Buffer

the RawBufferNavigator navigate in the raw data buffer and parse the header and send the payload as Buffer.

The buffer structure consists of :

  • the detector id (std::int32_t)
  • the datasource id (std::int32_t)
  • the event id (std::int32_t)
  • the bunch crossing id (std::int64_t)
  • the payload char array of (buffer size - 20) Based on
